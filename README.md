# vue3-ts-template

基于`vue3`和`typescript`的整合(架构)

## 安装与运行

### 安装依赖

```bash
npm i
```

### 运行

```bash
npm run dev
```

### 编译

```bash
npm run build
```

### 整理和修复文件

```bash
npm run lint
```

### 助手

```bash
yarn run helper -h
```

#### 开发模式

##### 依赖安装

```bash
yarn run helper:i
```

##### 调试

```bash
yarn run helper:dev
```

##### 助手编译

```bash
yarn run helper:build
```
